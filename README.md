# Екомкасса Чеки

Модуль помогает создавать чеки на заказы через облачную кассу **Екомкасса**.

## Работает в документах:
- Продажи. Заказы покупателей
- Продажи. Отгрузка
- Продажи. Возвраты покупателей

## Автоматизации:
- Создание чека продажи
- Создание чека возврата
- Создание 1-го и 2-го чека
- Списание маркировки

---

## Что такое Екомкасса?

**Екомкасса** — это платформа, которая решает задачи бизнеса, связанные с приемом платежей по 54-ФЗ за 1 минуту. Это экосистема решений на базе облачной кассы:

- Готовые решения для экономии на эквайринге до **2%** с помощью оплаты только СБП
- Приложение на смартфон для работы с кассой, платежами, ИИ-кассир и многое другое

---

## Преимущества:

### 1. Гарантия отсутствия штрафов
Мы — эксперты в 54-ФЗ. Подберем и настроим решение, которое будет соответствовать требованиям без рисков и штрафов.

### 2. Рыночные цены без скрытых платежей
Надежность каждому клиенту и качественный сервис за счет автоматизаций, надежного оборудования и высокоточного ПО для обработки очереди чеков.

### 3. Экосистема продуманных решений
К нам обращались клиенты, которым не хватало каких-то функций — мы включили это в экосистему **Екомкасса**. Мы удовлетворим **99,9%** запросов при развитии бизнеса и поможем это сделать быстрее.

### 4. Универсальный модуль "касса+платежи"
Объединили в одном окне **10+ банков** и интеграций приема платежей вместе с кассой. Если вы смените эквайринг — менять кассу не нужно. Сэкономим до **2%** на приеме платежей с помощью СБП.
